# Japan - Full Stack Development Projects

A collection of coding projects and learning materials including:

- **DSA** - Data Structures & Algorithms practice
- **Backend** - Backend development projects

## Quick Start

```bash
# Install dependencies for all projects
cd dsa && npm install
cd ../backend && npm install
```

## Project Structure

```
japan/
├── dsa/          # Data Structures & Algorithms
├── backend/      # Backend projects
└── README.md
```

## Development

Each project has its own `package.json` and scripts. Navigate to the specific project folder and refer to its documentation.
